# 🏦 Loans Data Analyst — Excel Project

A complete data analysis project built in Microsoft Excel
using a dataset of 200 loans.

## 📊 Project Structure

| Sheet | Description |
|-------|-------------|
| 📋 Raw Data | Original 200 loans with conditional formatting |
| 🔢 Calculated Data | EMI, Total Payment, Interest Paid using PMT() |
| 📊 Summary Statistics | COUNTIF, SUMIF, AVERAGEIF breakdown by loan type |
| 🔄 Pivot Analysis | Interactive Pivot Tables with Slicers |
| 📈 Charts | Bar, Pie and Line charts |
| 🏠 Dashboard | Executive KPI dashboard |

## 🛠️ Excel Skills Used

- PMT() formula for EMI calculation
- COUNTIF, SUMIF, AVERAGEIF
- Conditional Formatting
- Pivot Tables & Slicers
- Charts (Bar, Pie, Line)
- Dashboard Design

## 📁 Dataset

- 200 loans records
- Loan types: Car, Education, Home, Personal
- Columns: Loan ID, Customer ID, Loan Type,
  Loan Amount, Interest Rate, Tenure,
  EMI, Total Payment, Interest Paid, Default Flag

## 💡 Key Insights

- Total Portfolio: 200 loans
- Default Rate: ~19.5%
- 4 Loan Types analyzed
- Risk scoring for each loan
